# Changelog

## v1.0.0
- Local-first architecture intelligence for .NET solutions with zero source-code upload.
- Deterministic artifact generation in `./.archintel` for repeatable local and CI workflows.
- CI strict mode gates (`--strict`) for load issues and architecture violations.
- AI-friendly structured outputs (`scan_summary.json`, `projects.json`, `project_graph.json`, `namespaces.json`, `symbols.json`, `packages.json`, `docs.json`, `violations.json`) for downstream assistants.
