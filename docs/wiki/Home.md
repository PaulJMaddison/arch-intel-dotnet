## Repository at a glance
- **Solution entry point:** `ArchIntel.sln`
- **Primary source tree:** `src/`
- **Tests:** `tests/`
- **Build scripts:** `build.sh`, `build.ps1`

## Contribution workflow
1. Read the [CONTRIBUTING](../../CONTRIBUTING.md) guidelines.
2. Run the build script that matches your environment.
3. Add or update tests alongside your changes.

## Need help?
If you get stuck, start with the [Troubleshooting](../troubleshooting.md) guide or open an issue with
clear repro steps and logs.
